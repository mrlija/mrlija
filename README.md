Jsem student informační technologie se zaměřením na desktopové, mobilní a webové aplikace a multimédia na Střední průmyslové škole a Vyšší odborné škole v Liberci. Mám zkušenosti s designem herního prostředí, zajištění kvality, webovým designem a vývojem.
